# Collection Tools

Collection Tools is a libary of SillyTavern macros and slash commands for operations related to lists and dictionaries.

## Usage

For the list of available macros, send `/? macros` in the chat and scroll to the `List Utilities`, `Dictionary Utilities`(WIP), and `Collection Utilities` sections.

For the list of available slash commands, type `/nox-list`, `/nox-dict`, or `/nox-collection` in the chat input or QR IDE to see available commands and their documentation.

## Setup

### Prerequisites

*Extensions:*

- [Nox Library](https://github.com/CasualAutopsy/STLibs-Nox-Library)

### Installation

1. Make sure you've installed all the prerequisites.
2. Go to the extensions tab (stacked cubes icon) in SillyTavern and click the `Install extension` button.
3. Paste the following URL into the top input field: `https://github.com/CasualAutopsy/SillyTavern-Collection-Tools`.
4. Clcik `Install for all users` or `Install just for me`.
5. Done.
